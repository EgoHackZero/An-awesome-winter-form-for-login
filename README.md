# An Awesome Winter Form for Login

A beautiful, modern login form with a stunning winter theme and glassmorphism design effect. This project features a responsive login interface with smooth animations and an elegant frosted glass aesthetic.

## Features

- **Glassmorphism Design**: Modern frosted glass effect with backdrop blur
- **Winter Theme**: Beautiful winter background image creating a serene atmosphere
- **Responsive Layout**: Adapts seamlessly to different screen sizes
- **Smooth Animations**: Elegant label transitions on input focus
- **Form Validation**: Built-in HTML5 form validation
- **Icon Integration**: Ionicons for email and password fields
- **User-Friendly**: Includes "Remember Me" checkbox and "Forget Password" link
- **Registration Link**: Quick access to registration page

## Technologies Used

- HTML5
- CSS3
- [Ionicons](https://ionic.io/ionicons) - Icon library

## Installation

1. Clone the repository:
```bash
git clone https://github.com/EgoHackZero/An-awesome-winter-form-for-login.git
```

2. Navigate to the project directory:
```bash
cd An-awesome-winter-form-for-login
```

3. Open `index.html` in your web browser:

## Usage

Simply open the `index.html` file in any modern web browser. The form includes:

- **Email field**: Input with mail icon and validation
- **Password field**: Secure input with lock icon
- **Remember Me**: Checkbox to save user session
- **Forget Password**: Link to password recovery
- **Login Button**: Submit button with hover effect
- **Register Link**: Quick access to registration page

## Customization

### Changing the Background Image

Replace `winter.png` with your own image, or update the CSS in `style.css`:

```css
body {
    background-image: url('your-image.png');
}
```

### Adjusting the Glassmorphism Effect

Modify the backdrop blur in `style.css`:

```css
section {
    backdrop-filter: blur(55px); /* Adjust the blur value */
}
```

### Color Scheme

Change the border and text colors in `style.css` to match your brand:

```css
section {
    border: 2px solid rgba(255, 255, 255, 0.5); /* Adjust transparency */
}
```
